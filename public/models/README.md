# public/models/

Place patched ONNX models here for Pro Studio inference.

## yolov8n_webgpu.onnx

Generated by `scripts/export_webgpu_model.py`. Not committed to git (too large).

```
pip install onnx onnxsim
python scripts/export_webgpu_model.py
```

Then upload to Supabase Storage or CDN and update `YOLO_MODEL_URL_WEBGPU`
in `src/pages/admin/ProStudioBench.jsx` and `src/lib/proStudio/useOnnxInference.js`.
